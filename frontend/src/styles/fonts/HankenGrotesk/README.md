# Fonte — Hanken Grotesk

Cole aqui os arquivos estáticos da pasta `Hanken_Grotesk/static/` (e o `OFL.txt`).

Pesos recomendados para self-host via `next/font/local`:

- `HankenGrotesk-Regular.ttf` (400)
- `HankenGrotesk-Medium.ttf` (500)
- `HankenGrotesk-SemiBold.ttf` (600)
- `HankenGrotesk-Bold.ttf` (700)
- `HankenGrotesk-ExtraBold.ttf` (800) — opcional, para títulos/logo

Inclua também o `OFL.txt` (licença) nesta pasta.

> Alternativa: posso carregar via `next/font/google` (Hanken Grotesk existe no
> Google Fonts) e dispensar os arquivos locais. Me avise se preferir.
